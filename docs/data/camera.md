# Camera

How to add and convert photos into the storage.

## Folder structure

There are generally two file types to consider

  * Photo in JPG
  * Video in MP4

```
/camera
  /YYYY
    /Device
      /Event
        /image1.jpg
        /image1.xmp
    /Unknown
      /Event
        /image1.jpg
        /image1.xmp
```
